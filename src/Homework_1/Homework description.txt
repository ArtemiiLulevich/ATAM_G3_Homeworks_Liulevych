1. В отдельном классе А (название произвольное) создать:
- Массив (название любое) типа array ([]) с типом данных String и определёнными значениями
внутри (значения любые, количество больше 6)
- Метод с возвращаемым типом данных List<String> в котором для массива выше будет применён
цикл while, условие выхода из цикла – индекс
элемента массива, созданного в первом шаге == 6, при этом пропустить итерацию для элемента
с индексом 3 (не используем for, если нужно
создать внутри/снаружи какие либо переменные для выхода из цикла - создаём). Задача цикла
while – записать данные массива из шага 1 в
некий список ArrayList, который после работы цикла нужно вернуть на выходе из метода
(ключевое слово return)

2. В отдельном классе Б (название произвольное) создать:
- Метод с возвращаемым типом данных void, принимающим в себя переменную типа List<String>,
в котором с помощью цикла for будут выведены в консоль все элементы входящего в метод
массива.

3. В отдельном классе с названием Application создать метод main, который в процессе работы
должен вызвать все доступные методы выше созданных классов, а именно передать значение
отработки метода из класса А в метод класса Б.

Так же, не забываем использовать комментарии в коде для короткого объяснения как работает
тот или иной метод (для main комментарии не требуются, не комментируем каждую строку,
а только что делает тот или иной метод).Работу запаковать в архив (rar/zip) и сбросить
мне либо в телеграмм, либо на почту с указанием имени и номера группы.
